Nodal Domains of Eigenfunctions of Quantum Billiards
| Dr. Sudhir Jain, BARCOngoing

This project aims towards deriving recurrence relations over Number of Nodal Domains for Helmholtz-Eigenfunctions of Integrable Billiards - The 60-120° Rhombus, in particular.
This involves numerically solving the Helmholtz Equation using the Finite Element Method (FEM) and the Method of Fundamental Solutions (MFS) for the 60-120° rhombus using C++/MATLAB, calculating number of nodal domains using algorithms similar to those developed by Hoshen and Kopelman and finally Analysing the Data for existence of recurrence relations.

Rube Goldberg Machine | Prof. Sharat Chandran, IITBOngoing

A Rube Goldberg Machine, developed in C++ using the Box2D physics simulation engine.
The machine simulates perpetual machines like Somerset’s and Da Vinci’s machines, which are set in motion by a complex combination of springs, pendulum, pulleys and flowing fluids. Implemented features include dynamic frame rates and modification of Gravity for different bodies.

Sustenance | Microsoft code.fun.doSummer 2015

It is a Universal Windows App, a Strategic One-Player Game developed in C Sharp and XAML.
This Game led me to win the Microsoft code.fun.do Online Forum insert_emoticon

It uses Data Structures for different species, potions and industries and implements Differential Equations for Trophic Dynamics. The Player dispenses potions to species to counter-act the poison (dispensed by the Aritificial Intelligence) and prevent Extinction.
Sustenance portrays something as indispensable and vulnerable as the Environment in fun-loving way, while serving multiple purposes – Education, Awareness and Entertainment.

Digital Image Processing | Dr. Deepak B Phatak, IITBAutumn 2014

A Bitmap Image Editor with a C++ backend and an elegant gtkmm based user-friendly Graphical User Interface.
This Editor is a C++ only Application! Implemented features include Gaussian Blur, Local Histogram Equalization (Excessive Contrast-Enhancement), Sobel Edge Detection, Watermark Addition, among many others like Image Warping, Dither, Halftoning and Hue Modification.
